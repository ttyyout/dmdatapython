# DMDATA 구독 전문 전체 목록

## 緊急地震（予報）区分

1. **緊急地震速報配信テスト** (VXSE42)
   - 긴급지진속보 테스트 전문
   - 처리: 무시 (테스트용)

2. **緊急地震速報(予報)** (VXSE44)
   - 긴급지진속보 예보
   - 처리: VXSE45와 동일하게 처리

3. **緊急地震速報(地震動予報)** (VXSE45)
   - 긴급지진속보 경보 (지진동 예보)
   - 처리: EEW 처리 로직 사용

## 地震・津波関連 区分

### 해일 관련
1. **地震·津波に関するお知らせ** (VZSE40)
   - 지진・해일 관련 안내
   - 처리: 안내 정보 처리

2. **津波警報·注意報·予報** (VTSE41)
   - 해일 경보・주의보・예보
   - 처리: 해일정보 처리 로직 사용

3. **津波情報** (VTSE51)
   - 해일 정보
   - 처리: 해일정보 처리

4. **沖合の津波情報** (VTSE52)
   - 외해 해일 정보
   - 처리: 해일정보 처리

5. **国際津波関連情報(国内向け)** (WEPA60)
   - 국제 해일 관련 정보 (국내용)
   - 처리: 국제 해일정보 처리

### 지진 정보
6. **震度速報** (VXSE51)
   - 진도속보
   - 처리: 지진상세정보 처리 (report_type: "sokuhou")

7. **震源に関する情報** (VXSE52)
   - 진원정보
   - 처리: 지진상세정보 처리 (report_type: "epicenter")

8. **震源·震度に関する情報** (VXSE53)
   - 진원・진도정보
   - 처리: 지진상세정보 처리 (report_type: "detail")

9. **地震の活動状況等に関する情報** (VXSE56)
   - 지진 활동 상황 등 관련 정보
   - 처리: 지진 활동 정보 처리

10. **地震回数に関する情報** (VXSE60)
    - 지진 횟수 관련 정보
    - 처리: 지진 횟수 정보 처리

11. **顕著な地震の震源要素更新のお知らせ** (VXSE61)
    - 현저한 지진의 진원 요소 갱신 안내
    - 처리: 진원 요소 갱신 처리

12. **長周期地震動に関する観測情報** (VXSE62)
    - 장주기 지진동 관련 관측 정보
    - 처리: 장주기 지진동 정보 처리

13. **推計震度分布図作図用データ** (IXAC41)
    - 추정 진도 분포도 작도용 데이터
    - 처리: 진도 분포 데이터 처리

### 남해해구 지진 관련
14. **南海トラフ地震臨時情報** (VYSE50)
    - 남해해구 지진 임시 정보
    - 처리: 남해해구 지진 정보 처리

15. **南海トラフ地震関連解説情報(定例外)** (VYSE51)
    - 남해해구 지진 관련 해설 정보 (정례 외)
    - 처리: 남해해구 지진 해설 정보 처리

16. **南海トラフ地震関連解説情報(定例)** (VYSE52)
    - 남해해구 지진 관련 해설 정보 (정례)
    - 처리: 남해해구 지진 해설 정보 처리

17. **北海道·三陸沖後発地震注意情報** (VYSE60)
    - 홋카이도・산리쿠 해역 후발 지진 주의 정보
    - 처리: 후발 지진 주의 정보 처리
